# Smart Contact Manager

A web-based Contact Management System built using Spring Boot that allows users to securely manage their contacts with role-based authentication and authorization.

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- Spring Data JPA

### Frontend
- JSP
- HTML
- CSS
- Bootstrap

### Database
- MySQL

### Build Tool
- Maven
